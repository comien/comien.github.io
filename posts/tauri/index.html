Tauri

Posted by 雾雨七辰 on Wed, May 17, 2023

我个人使用了一段时间 Rust，起初比较难入手，有些劝退。但是了解 Rust 一些特性并且能写一些功能的之后，喜欢上了 Rust 代码的舒适度。
Tauri 是用 Rust 写的桌面应用程序框架，可与 Electron 媲美。 因为我不熟悉 Vue，所以我的代码中使用 react 作为 UI 部分。
Tauri 官方文档

getting started

开始写代码前，需要安装 cli 脚手架工具，帮助我们创建项目。
Tauri 脚手架兼容了多数前端栈，因为 Tauri 逻辑需要 Rust，所以我选择 cargo 安装 cli。 create-tauri-app 帮助我们创建项目框架模板。
我们借助 tauri-cli 调试应用。

1cargo install create-tauri-app
2cargo install tauri-cli

创建项目

安装完成后，我们可以借助 create-tauri-app 创建项目框架。

1# my-tauri-app 项目名
2# --template react 前端框架
3cargo create-tauri-app my-tauri-app --template react --manager pnpm
4
5cd my-tauri-app
6# 把依赖安装上
7npm install
8# 启动应用
9cargo tauri dev

这样 tauri 的模板应用就启动了
